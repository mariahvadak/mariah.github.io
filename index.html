Pixel Validations

Center for Earth Observation and Modeling

For this project I was given the task of modifying a colleague's pixel validation map to show a lightly highlighted polygon on areas where there was available data.

If you look closely, you can see tiny points that represent data locations. Each data pixel is super small at this zoom level!

Zooming in will give you more clear image of my implementation. Now users can see different pixel sizes of data which can be used in other parts of the website to categorize pixels!

If you want to see my project in action on the live webpage, then go ahead and check out the Center for Earth Observation and Modeling(CEOM) website!
